## Platform Capabilities

### Technology Stack
- **Frontend**: HTML5, CSS3, JavaScript (no frameworks!)
- **AI Engines**: Stockfish.js (WASM), custom algorithms
- **Multiplayer**: WebRTC + Firebase/Supabase
- **Storage**: IndexedDB + LocalStorage
- **APIs**: Web Speech API, Chess.com, Lichess
- **Total Size**: ~10MB (without optional Lc0)

### Browser Requirements
- Modern browser (Chrome 90+, Firefox 88+, Safari 14+)
- WebAssembly support
- Web Workers support
- Speech API (for tongue twisters)
- WebRTC (for real-time multiplayer)

### Works Offline
- All single-player games
- AI opponents
- Local multiplayer
- Settings and stats
- **Only needs internet**: Online multiplayer, API features
