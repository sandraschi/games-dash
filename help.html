Option 1: Simple Start (Recommended)

  1. Open PowerShell in the games-app folder
  2. Run: .\START_EVERYTHING.ps1
  3. Browser opens automatically at http://localhost:9876

Option 2: Docker

  1. Make sure Docker Desktop is running
  2. Run: docker compose up -d
  3. Open http://localhost:9876

üåê Multiplayer (Optional)

‚ö†Ô∏è Firebase Setup Required

Multiplayer requires Firebase configuration. This is optional - all games work without it!

If you want multiplayer:

  1. Create a Firebase project at console.firebase.google.com
  2. Enable Authentication (Email/Password)
  3. Enable Realtime Database
  4. Copy your config to firebase-config.js
  5. See FIREBASE_SETUP_GUIDE.md for detailed steps

Without Firebase: All games work perfectly in single-player or local multiplayer mode!

‚öôÔ∏è Troubleshooting

AI doesn't work in Chess/Shogi/Go

Make sure the backend servers are running:

.\START_EVERYTHING.ps1

Or start manually:

python stockfish-server.py
python shogi-server.py
python go-server.py

Games not loading

Make sure web server is running on port 9876:

python -m http.server 9876

Tongue twisters sound wrong

Install language packs:

Windows Settings ‚Üí Time & Language ‚Üí Speech ‚Üí Add voices

Docker issues

Rebuild containers after code changes:

docker compose down
docker compose build
docker compose up -d
